# 💘 Bot Znakomstva

Телеграм бот для знакомств на aiogram 3 + PostgreSQL

## Функционал
- 📝 Создание и редактирование анкеты
- 🔍 Поиск новых людей
- 💖 Лайки и матчи
- 👤 Просмотр и удаление матчей

## Установка

1. Клонируй репозиторий
2. Установи зависимости
    pip install -r requirements.txt
3. Создай .env файл
    BOT_TOKEN=твой_токен
    DB_NAME=znakomstva
    DB_PASSWORD=пароль
4. Запусти бота
    python main.py
