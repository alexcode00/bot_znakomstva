# First Parser

This is my first web scraping project.
I used [Books to scrape](https://books.toscrape.com/) for training.

## Technologies
- Python 3.x
- requests
- BeautifulSoup (bs4)
- lxml

---

## How to run

1. Clone the repository:
```bash
   git clone https://github.com/afkgho2/my-first-parser.git
   cd my-first-parser
```
2. Install dependencies:
```pip install -r requirements.txt```
3. Run the script:
```python first_parser.py```
